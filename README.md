# 🎓 Face Recognition Attendance Monitoring System

A **Python-based desktop application** that automates student attendance using **real-time face recognition**. The system detects and recognizes registered students through a webcam, records attendance with timestamps, and stores the data in CSV format. It features a simple and user-friendly graphical interface built with **Tkinter**.

---

## 📌 Project Overview

Traditional attendance systems are time-consuming and prone to manual errors. This project leverages **Computer Vision** and **Face Recognition** to streamline the attendance process by automatically identifying registered individuals and recording their attendance.

---

## ✨ Features

* Real-time face detection and recognition
* Interactive Tkinter-based GUI
* Student registration with image capture
* Face dataset generation and model training
* Automatic attendance recording with date and time
* Daily attendance reports in CSV format
* Password-protected student registration
* Student details management
* Easy-to-use desktop application

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **GUI Framework:** Tkinter
* **Computer Vision:** OpenCV
* **Face Recognition:** LBPH Face Recognizer
* **Libraries:** NumPy, Pandas, Pillow
* **Data Storage:** CSV Files

## ⚙️ Installation

### Clone the repository

```bash
git clone git clone https://github.com/Saahini/face-recognition-attendance-system.git
```

### Navigate to the project folder

```bash
cd face-recognition-attendance-system
```

### Install dependencies

```bash
pip install -r requirements.txt
```

### Run the application

```bash
python main.py
```

---

## 🚀 How to Use

1. Launch the application.
2. Register a new student by entering the required details.
3. Capture facial images for the student.
4. Train the face recognition model.
5. Start the attendance module.
6. The system automatically recognizes registered faces.
7. Attendance is recorded with the current date and time.
